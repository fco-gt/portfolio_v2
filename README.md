# Portafolio creado con Astro y Tailwind CSS

## Portafolio para FCO-GT

### Instrucciones de uso:
- Clonar el repositorio
```bash
    git clone https://github.com/fco-gt/portfolio
```

- Instalar dependencias
```bash
    npm install
```

- Iniciar servidor de Astro
```bash
    npm run dev
```

- Visita tu localhost
http://localhost:4321/
